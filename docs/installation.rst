Installation
============

.. toctree::
    :hidden:

``sounds-deep`` can be installed from GitHub via ``pip``
::

    $ pip install -e https://github.com/DrKwint/sounds-deep.git

If you are having trouble with this step, make sure ``pip`` is up to date by
running
::

    $ pip install --upgrade pip

Prerequisites
-------------

``sounds-deep`` currently only supports Python 3.6.
